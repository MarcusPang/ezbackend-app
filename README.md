# talkMore

Social Media application built using ezbackend

## Features

Users can:

- post and comment under post (post text, images, videos)
- like posts and comments
- befriend each other
- chat with other users
- follow other users
- join communities and chat with other users in a group chat

## Tech Stack

### Frontend

- Next.js
- TypeScript
- Tailwindcss

### Backend

- Ezbackend
- TypeScript
